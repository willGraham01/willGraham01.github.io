This is a paragraph. Within a paragraph you can use various elements to change the appearance of text; for example give emphasis to certain words or phrases, draw attention to keywords using italics, or underline words, which you might also want to do in your headings. You can also change the mid-sentence if you want, or the actual font that your text is rendered in.
Newlines in the HTML source file are not rendered, you need to use the break element to force a newline to be displayed. If you want your entire paragraph to have the same formatting, for example you want your paragraph to have red text and be in courier font throughout, you can edit the style attributes of that element. This paragraph is formatted differently to the others. This is because when I specified the element, I changed some of it's styles from the defaults. This can be useful if you only have a few paragraphs that require different formatting, but in general you'll want to use a style sheet to standardise the appearence of your elements.

I'm now going to list some of the attributes that a paragraph's style can have, using an unordered list:

  • color: for font colour (damned americanisation). Either give a colour keyword, or a hex code
  • font-family: font of this paragraph
  • background-color: gives a block background colour to this paragraph
  • font-size: either an amount in pixels (px) or as a percentage of the default body text size
  • text-align: alignment of text; either left, right, or center.
This is a non-exhaustive list, and a google search will allow you to find all the other attributes that you can play around with.

Images and Links

You can add images and links to your webpage using two further elements. You will need to know either the directory and filename of what you are linking to (for example, if you are linking to another page on your website or to an image stored in your public_html folder); or the web address of the target if linking to something online (like an external page). You can also link to documents and other file types, using the same syntax. If you want links to open in a new tab, you need to add the target attribute to your link elements.
For example:

  • You can go to my homepage, which is stored in the same place as this webpage. The file is called index.html.
  • You can go to my homepage again, but this time it will open in a new tab.
  • You can one of my tutorial notes files, which will prompt your browser to download the file.
  • You can go to the SAMBa website, in a new tab. Notice how you need the http:// in the web address!

Tables

Finally, let's talk about tables - useful for designing schedules, like that for SLS maybe? Tables have their own element, as well as elements to specify new rows, columns, and headings.

Notice how this table doesn't look every fancy, so we will need to add some styling to it. We could do all this by editing the style attribute of the table above, but a much better idea for us will be to use a Cascading Style Sheet (.css) to define our own class of table, and then link this table to that class!
